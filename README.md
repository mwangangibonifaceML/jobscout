## GMail Job Agent
AI-powered multi-agent system that

    - Connects to email
    - Retrieves email and classifys them
    - Detects job oppotunities sent to the email.
    - Extracts job details.
    - Reads CVs
    - Matches jobs against user skills.
    - Generates personalized recommendations.

## Tech Stack

    - Python
    - Google-ADK
    - Gemini
    - ChromaDB
    - Gmail API
    - Streamlit
    - SQLAlchemy
